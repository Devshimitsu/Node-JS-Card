### Installation For Linux

- Just, Clone this repository -
```
git clone https://github.com/Devshimitsu/Node-JS-Card.git
```

- Change to cloned directory  -
```
$ cd Node-JS-Card
```
```
$ npm install
```
```
$ npm start
```


### Installation For windows

- Just, Download this repository 
- open cmd and type 
```
npm install 
```
```
npm start
```
